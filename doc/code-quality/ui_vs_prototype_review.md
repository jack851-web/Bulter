# Bulter UI 设计与产品原型对照审查

> 对照 `doc/first/figures/phone-*.png` 共 20 张产品原型图，逐页核对当前实现，列出偏差并标注严重级别。
> 审查日期：2026-06-17
> 审查范围：`src/lib/` 全部 UI 代码

## 严重等级定义

| 等级 | 含义 |
|---|---|
| 🔴 P0 | 关键页视觉与原型严重偏离，用户一眼能看出"不像 Bulter" |
| 🟠 P1 | 模块 / 状态 / 交互不正确，影响日常使用 |
| 🟡 P2 | 微调：间距 / 字号 / 颜色 / 文案与原型不一致 |

---

## 1. Butler 中枢主页（原型：phone-01-home.png）

| 项 | 原型 | 当前实现 | 等级 |
|---|---|---|---|
| 整体布局 | **5 张全宽模块卡纵向堆叠**（关系 / 成长 / 财富 / 思想 / 健康）| 1 张 AI 洞察大卡 + 2×2 网格 4 张小卡 | 🔴 P0 |
| 卡片底色 | **每个模块卡背景是模块品牌色的低透明度色块**（如关系是粉、成长是墨绿、财富是金、思想是紫、健康是绿）| 统一为白底，仅左侧 4px 品牌色细条 | 🔴 P0 |
| 卡片内信息结构 | "Butler · 今日" 或模块名 + 一句 AI 洞察 + 底部 1-2 个小计数胶囊 | 纯标题 + 副标题 | 🔴 P0 |
| 标题字段 | "下午好，小明" / "今天 · 3 件事待已入栈，4 件事待呈" | "今天想做点什么？" | 🟡 P2 |
| 顶部 AI 大卡 | 橙色 Butler 卡片，下方 2 个并列胶囊 ("2 个待呈", "1 个待人") | 居中 AI 洞察卡，无计数胶囊 | 🟠 P1 |

### 修复方案
- 改 `butler_home_page.dart` 为 5 张全宽模块卡，每张卡展示：
  - 卡顶一行：模块名胶囊（左上角）+ "今日" 副标签
  - 卡中央：AI 总结 / 关键数据（如 "5 个联系人 / 2 个待联系"）
  - 卡底：1-2 个事实胶囊（"待联系 2 位"、"人情 1 笔未还"）
  - 卡底色：模块品牌色 alpha 0.10 ~ 0.18

## 2. AI 对话页（原型：phone-02-ai-chat.png）

| 项 | 原型 | 当前实现 | 等级 |
|---|---|---|---|
| AI 气泡位置 | **左侧**、浅米色底、圆角矩形 | 右侧 / 黑色 CTA / 圆角矩形 | 🔴 P0 |
| 用户气泡位置 | **右侧**、纯黑底、白字 | 左侧 / 浅灰底 | 🔴 P0 |

| 中段统计卡 | 3 张并列白卡：总支出 ¥4,820 / 餐饮 ↑18% / 预算剩余 ¥680 | 无 | 🟠 P1 |
